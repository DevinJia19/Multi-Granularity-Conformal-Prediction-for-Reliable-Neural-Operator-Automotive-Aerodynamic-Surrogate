评估指标说明：
  • MAE (平均绝对误差)：
    定义: mean(|预测值 - 真实值|)
    含义: 预测结果平均偏离真实值的程度
    目标: 越小越好（< 0.02 表示优秀）
  
  • RMSE (均方根误差)：
    定义: sqrt(mean((预测值 - 真实值)²))
    含义: 对大误差更敏感的误差度量
    目标: 越小越好（< 0.02 表示优秀）
  
  • MAPE (平均绝对百分比误差)：
    定义: mean(|预测值 - 真实值| / |真实值|) × 100%
    含义: 相对误差百分比
    目标: < 5% 表示优秀，< 10% 表示可接受
  
  • R² (决定系数)：
    定义: 1 - SS_res / SS_tot
    含义: 模型解释数据方差的比例
    范围: -∞ ~ 1
    目标: > 0.98 表示优秀，> 0.95 表示很好
  
  • Correlation (皮尔逊相关系数)：
    含义: 预测值和真实值的线性相关程度
    范围: -1 ~ 1
    目标: > 0.99 表示优秀，> 0.98 表示很好

🔧 常见调整和优化
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

问题1：GPU内存不足 (CUDA out of memory)
  解决方案: 在 config.py 中降低 BATCH_SIZE
    从 8 改为 4 或 2
  
  # 在 config.py 中修改
  class TrainingConfig:
      BATCH_SIZE = 4  # 改小

问题2：训练速度太慢
  解决方案:
    • 增加 BATCH_SIZE (如果GPU内存足够)
    • 增加 NUM_WORKERS 数据加载线程
    • 降低 NUM_POINTS 点云点数
  
  class TrainingConfig:
      BATCH_SIZE = 16  # 增大
      
  class DataConfig:
      NUM_WORKERS = 8  # 增加
      NUM_POINTS = 8192  # 降低

问题3：预测精度不好 (MAPE > 10%)
  解决方案:
    • 增加 NUM_EPOCHS 训练轮数
    • 降低 LEARNING_RATE 学习率
    • 增加 N_LAYERS 网络深度
    • 增加 N_HIDDEN 隐藏维度
  
  class TrainingConfig:
      NUM_EPOCHS = 200  # 增加
      LEARNING_RATE = 0.0005  # 降低
  
  class ModelConfig:
      N_LAYERS = 6  # 增加
      N_HIDDEN = 384  # 增加

问题4：模型过拟合 (训练损失小，但测试精度差)
  解决方案:
    • 增加 DROPOUT 比例
    • 增加 WEIGHT_DECAY 权重衰减
    • 降低网络复杂度
  
  class ModelConfig:
      DROPOUT = 0.2  # 增加
  
  class TrainingConfig:
      WEIGHT_DECAY = 1e-4  # 增加

使用预设配置快速切换：
  # 在 train.py 中使用
  from config import PresetConfigs
  config = PresetConfigs.get_high_accuracy()  # 高精度
  config = PresetConfigs.get_fast_training()   # 快速训练

  输入:   点云 (B, 16384, 3)  # 每个模型16384个3D点

模型架构:
  主干:   GeoTransolver
          • 4层 Transformer blocks
          • 256维隐藏层
          • 8个注意力头
  回归头: 3层MLP (256→64→32→1)

训练设置:
  优化器: Adam (lr=0.001)
  损失:   MSE (回归)
  调度:   Cosine Annealing
  Epoch:  100

数据分割:
  训练集: 80% (~3200样本)
  测试集: 20% (~800样本)

预期结果:
  MAE:   < 0.02
  MAPE:  < 5%
  R²:    > 0.98
